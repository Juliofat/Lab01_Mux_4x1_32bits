# Lab01 - Multiplexador 4x1 (32 bits)

Implementação de um multiplexador 4:1 de 32 bits em SystemVerilog.

## Arquivos

- `Mux_4x1_1b.sv` → implementação do multiplexador de 1 bit
- `Mux_4x1_32b.sv` → multiplexador de 32 bits usando generate
- `Tb_Mux_4x1_32b.sv` → testbench para verificação do circuito

## Simulação

A simulação foi realizada utilizando ModelSim.
O testbench gera entradas aleatórias e verifica automaticamente se a saída está correta.
